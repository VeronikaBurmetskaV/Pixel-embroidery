# Піксельна вишивка. Редактор орнаменту
## Розробити програму для створення схеми української вишивки.
### Вимоги до програми, обов'язкові функції:
- Користувач повинен мати змогу інтерактивно обирати кольори з палітри та малювати на сітці клітинок, формуючи власний візерунок. Дані вишивки зберігаються у внутрішній структурі (масив або список).
- При відкритті програми автоматично генерується схема вишивки Вашого імені.
- Додайте можливість збереження та відкриття схем (наприклад, у форматі PNG).
- Додайте можливість автоматично будувати схему вишивки з симетрією (горизонтальною, вертикальною), і з дублюванням.
- Користувач задає розміри, кольори, основний фрагмент орнаменту, який автоматично дублюється з обраною симетрією.
- У заголовку вікна програми є назва та автор.
